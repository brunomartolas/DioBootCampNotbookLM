# DioBootCampNotbookLM
Desafio Bootcamp Bradesco

Contexto e Objetivos 

Esse repositório se refere ao desafio do BootCamp do Bradesco, disponivilizado pela platafor Dio.me, para explorar o poder do NotebookLM para criação de conteúdo.
O tema escolhido foi o firewall Pfsense, que é uma ferramenta OpenSource, com vários recursos em relação a monitoramente de vulnerabilidades e controle de ameaças cibernéticas, sendo assim, o objeto do estudo
